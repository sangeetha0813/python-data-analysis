# Pandas Sales Data Analysis

A hands-on **Pandas** practice project built using a sample sales dataset. This repository contains **50+ practical exercises** covering most of the Pandas concepts commonly used in data analysis.

The goal of this project is to strengthen my understanding of:

- Data manipulation
- Data filtering
- Aggregation
- GroupBy operations
- Pivot tables
- Business analysis using Python and Pandas

---

##  Dataset

**File:** `sales_data.csv`

The dataset contains the following information:

- Order ID
- Customer
- City
- Category
- Product
- Price
- Quantity
- Sales
- Discount
- Payment Mode
- Order Status

- Python 3.x
- Pandas
- Jupyter Notebook

#  Topics Covered

## 1. Data Inspection

Learn how to inspect and understand a dataset.

Topics include:

- Loading CSV files
- Viewing rows
- Checking dimensions
- Viewing column names
- Data types
- Summary statistics
- Missing values

---

## 2. Data Filtering

Practice selecting data using different conditions.

Examples include:

- Filter by City
- Filter by Category
- Filter by Status
- Multiple conditions
- Numeric filtering
- Missing values

---

## 3. Sorting Data

Learn how to sort datasets by one or multiple columns.

Examples include:

- Highest sales
- Cheapest products
- Multi-column sorting

---

## 4. Column Operations

Practice creating and modifying columns.

Topics include:

- Renaming columns
- Creating GST column
- Calculating Final Amount
- Data type conversion
- String ↔ Integer conversion

---

## 5. Aggregations

Perform numerical analysis using:

- Sum
- Mean
- Minimum
- Maximum
- Count
- Unique counts

---

## 6. GroupBy Operations

Learn one of the most powerful features in Pandas.

Examples include:

- Total Sales by City
- Average Price by Category
- Total Quantity sold by Category
- Orders by Status
- Total Sales by Payment Mode

---

## 7. Pivot Tables

Summarize large datasets using Pivot Tables.

Examples include:

- Total Sales by City
- Average Price by Category
- City vs Category Sales
- Payment Mode vs Status Quantity

---

## 8. Missing Values

Practice handling incomplete data.

Examples include:

- Finding missing values
- Replacing missing values
- Removing missing rows

## 9. Frequency Analysis

Examples include:

- Orders by Category
- Customers by City
- Most frequent Product
- Most popular Payment Mode
